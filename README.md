# Yashica Mittal - Portfolio

A modern, interactive portfolio website showcasing AI/ML research, web development projects, and design work.

## Features

- 🎨 Custom neural OS-inspired design system
- 🌊 Fluid mesh interactive background
- 📊 Real-time skill radar visualization
- 🖥️ Terminal-style contact interface
- 📱 Fully responsive across devices
- ✨ Custom cursor and animations

## Tech Stack

- Pure HTML5, CSS3, JavaScript (Vanilla)
- Canvas API for interactive graphics
- Intersection Observer for scroll animations
- No external dependencies

## Deploy to Vercel

### Quick Deploy

1. **Install Vercel CLI** (if not already installed):
   ```bash
   npm install -g vercel
   ```

2. **Deploy**:
   ```bash
   cd path/to/project
   vercel
   ```

3. Follow the prompts to link your project and deploy.

### Deploy via GitHub

1. Push this project to a GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

### Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New..." → "Project"
3. Drag and drop this folder or select it from your file system
4. Click "Deploy"

## Local Development

To preview locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Then open http://localhost:8000 in your browser
```
